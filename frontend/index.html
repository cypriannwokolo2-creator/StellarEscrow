Help Center

API Documentation

The StellarEscrow Indexer provides a RESTful API for querying contract events. All endpoints return JSON responses and support standard HTTP methods.

GET /health

Check the health status of the indexer service.

Example Response

{
  "status": "healthy",
  "timestamp": "2024-01-15T10:30:00Z"
}
GET /events

Retrieve a paginated list of contract events with optional filtering.

event_type
Filter by event type (optional)
trade_id
Filter by trade ID (optional)
limit
Maximum results (default: 50, max: 100)
offset
Pagination offset (default: 0)
GET /events/{id}

Retrieve a specific event by its UUID.

GET /search

Global search across trades, users, and arbitrators.

q
Search query string (required)
limit
Maximum results per category (default: 10)
GET /search/trades

Search specifically within trades.

q
Search query (optional)
status
Filter by trade status (optional)
seller
Filter by seller address (optional)
buyer
Filter by buyer address (optional)
GET /ws

WebSocket endpoint for real-time event streaming. Connect to receive live updates as events are indexed.

API Accessibility Features

  • All responses include human-readable error messages
  • Event types are provided in both machine-readable (snake_case) and human-readable formats
  • Timestamps follow ISO 8601 format for consistent parsing
  • Rate limiting headers indicate remaining requests for planning
  • CORS support enables cross-origin API access
